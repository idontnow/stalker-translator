S.T.A.L.K.E.R. XML Translator Tool
Version: 54.0 (The Survivor Release)
==================================

This is a versatile tool designed for translating S.T.A.L.K.E.R. series games (X-Ray engine).
It is designed to run anywhere, even with minimal dependencies.

DEPENDENCIES:
-------------
1. Python 3.x (Mandatory)
2. Optional Libraries (Recommended but not required):
   - beautifulsoup4 : Robust XML parsing (Tool uses a basic fallback without it).
   - deep-translator : Enables Google Translate feature.
   - colorama       : Enables colored UI in terminal.
   - google-genai   : Enables AI translation (Gemini).

   Install all with:
   pip install beautifulsoup4 deep-translator colorama google-genai

GETTING STARTED:
----------------
1. Run the Setup Wizard:
   python setup_project_Public.py

2. Start Translating:
   python stalker_translator_Public.py

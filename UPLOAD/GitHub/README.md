---

## âœ¨ Features

* **Project Based:** Manage multiple translations (SoC, CS, CoP, Mods) separately.
* **Smart Navigation:** Remember exactly where you left off in each file.
* **Automatic Backups:** Keeps your `Original_Backup` safe and creates snapshots of your work.
* **AI Power:** Optional integration with **Google Translate** and/or **Google Gemini AI** for more context-aware translation.
* **Resilience:** Works even without dependencies (in B&W mode).
